# Issue 28.1.15 - Streaming: Network Type Change (WiFi ↔ Cellular)

**Status:** OPEN
**Priority:** Medium
**Category:** Feature / Testing
**Created:** 2026-02-16

## Description

Implement and test seamless streaming continuation when the device switches between WiFi and cellular networks.

## Spec Scenarios (from `spec/streaming-playback.md`)

1. **WiFi → Cellular** — streaming continues without interruption or re-buffering
2. **Cellular → WiFi** — streaming continues seamlessly

## Acceptance Criteria

- [ ] Playback continues without user-visible interruption during network type change
- [ ] No duplicate re-buffering when switching from WiFi to cellular
- [ ] Integration test simulates network type transition and verifies continuity

# Issue 12.8.6: Increase Parallelism for Non-UI Suites

## Priority
Low

## Status
Open

## Description
Unit and integration jobs currently run serially with a single worker. Enabling `-maximum-parallel-testing-workers` for non-UI bundles could shrink overall pipeline time without affecting UI determinism. Preflight already builds artifacts, so added parallelism should mostly affect test execution.

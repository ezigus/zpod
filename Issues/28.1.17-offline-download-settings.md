# Issue 28.1.17 - Offline: Download Settings (Wi-Fi Only, Cellular Warning, Quality)

**Status:** OPEN
**Priority:** Low
**Category:** Feature
**Created:** 2026-02-16
**Parent Issue:** 28.1

## Description

Implement download preference settings: Wi-Fi only mode, cellular data warning prompt, and download quality selection.

## Spec Scenarios (from `spec/offline-playback.md`)

1. **Wi-Fi Only setting** — downloads only start on Wi-Fi; queued on cellular
2. **Cellular data warning** — prompt user before downloading over cellular
3. **Download quality selection** — user can choose audio quality (standard/high)

## Acceptance Criteria

- [ ] Wi-Fi Only toggle in download settings prevents cellular downloads
- [ ] Cellular warning alert appears with proceed/cancel options
- [ ] Quality selection persists and affects download URL selection
- [ ] Settings integrate with DownloadCoordinator queue logic

## Dependencies

- Requires: Issue 28.1 (download infrastructure), SettingsDomain
